# zkVoting Notification Service API
> 🌟 대량 알림 발송을 위한 내부 알림 마이크로서비스 입니다.

#### Installation

<!-- > [!IMPORTANT]
> 해당 프로젝트는 `pnpm`을 사용하고 있습니다. `postinstall`에 관련 스크립트를 설정했으니 참고 부탁드립니다. -->

```bash
$ npm install
```

#### Before Start

```bash
# For development: 개발 환경은 일반 nodemon에서 돌아갑니다.
$ npm run start:dev

# For production: 프로덕션 환경은 container 환경을 가정하며, pm2를 사용하여 구동을 합니다.
$ npm run start:prod
```

#### Test

```bash
# unit tests
$ npm run test
```
